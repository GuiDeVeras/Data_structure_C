# Project's struct
- `static_p_queue.h`: Definition of the data type stored in the queue, the pointer "p_queue", the size of the array and function prototypes.
- `static_p_queue.c`: Definition of the data types "p_queue", implementation of the functions.
	- `main.c`: Management of the user interface and input/output (I/O) operations.
- `static_p_queue_test.c`: Implementation of the test functions.
	- `main_test.c`: Unitary test for validate the creation and manipulation.

	The purpose of this project is to learn the applications of static priority queue.
	
• Static priority queue: A type of "queue" that generalize the idea of sorting. The inserted elements have an extra data: their priority. This value detemines the elements queue position and also the first to be removed.
