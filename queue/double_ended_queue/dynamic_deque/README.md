# Project's struct
- `dynamic_deque.h`: Definition of the data type stored in the queue, the pointer "deque", the size of the array and function prototypes.
- `dynamic_deque.c`: Definition of the data types "deque", implementation of the functions.
	- `main.c`: Management of the user interface and input/output (I/O) operations.
- `dynamic_deque_test.c`: Implementation of the test functions.
	- `main_test.c`: Unitary test for validate the creation and manipulation.

	The purpose of this project is to learn the applications of double ended queue.
	
• Double ended queue (deque): A type of "queue" that allows insertion and remotion in both ends.

• Dynamic deque: Your implementation utilises the same structure as sequential dynamic queue. It's treated like "circular". Utilises an array to store the elements
