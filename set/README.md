# Project's struct
- `set.h`: Definition of the data type stored in the queue, the pointer "s" and function prototypes.
- `set.c`: Definition of the data types "s", implementation of the functions.
	- `main.c`: Management of the user interface and input/output (I/O) operations.
- `set_test.c`: Implementation of the test functions.
	- `main_test.c`: Unitary test for validate the creation and manipulation.

	The purpose of this project is to learn the applications of set.
	
• Set: A data structure utilised for store and organize data, without repetitions and without any particular order. The computational implementation follows the mathematical concept of a finite set. It can be implemented using a hash table or a search binary tree.
