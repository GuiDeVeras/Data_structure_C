# Project's struct
- `bag.h`: Definition of the data type stored in the queue, the pointer "b" and function prototypes.
- `bag.c`: Definition of the data types "b", implementation of the functions.
	- `main.c`: Management of the user interface and input/output (I/O) operations.
- `bag_test.c`: Implementation of the test functions.
	- `main_test.c`: Unitary test for validate the creation and manipulation.

	The purpose of this project is to learn the applications of multiset.
	
• Multiset (bag): A data structure similar to a shopping cart, in which you can add one or more occurrences of an item, not caring with the order. Can be implemented using a static array or a linked dynamic list.
